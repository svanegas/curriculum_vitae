## How to compile
#### Requirements
A full TeX distribution is assumed.  I used [MacTex](http://www.tug.org/mactex/mactex-download.html) for OS X.

#### Usage
At a command prompt, run
```bash
$ xelatex resume.tex
```
This should result in the creation of ``resume.pdf``
